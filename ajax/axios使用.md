## 创建项目
author 作者  
npm 包管理工具(管理我们下载的各种资源)
npm init npm初始化   会生成一个package.json文件 我们可以写脚本，保存我们下载的第三方包 
npm init -y  一键初始化
package.json 里面 script 脚本里面配置的命令 会对应npm 的命令  脚本可以自己任意配置 但是注意你的命令要存在 
 "start": "node server.js" 
 npm start = node server.js
